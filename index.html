FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers reaslistic coding challenges to help developers improve their frontend coding skills with projects in HTMl, CSS, and Javascript. It's suitable for all levels and ideal for portfolio building

Is Frontend Mentor Free?

Yes, Frontend Mentor is free, offering web development challenges in HTML, CSS, and JavaScript. Frontend Mentor Pro, a paid option, provides additional features for skill enhancement.

Can i use Frontend Mentor projects in my portfolio?

Certainly! You have the freedom to feature Frontend Mentor projects prominently in your portfolio, showcasing your skills and accomplishments in web development.

How can i get help if i'm stuck on a challenge?

If you're stuck on a challenge, you can seek help through the Frontend Mentor community forum or access their Discord channel for assistance and guidance.
