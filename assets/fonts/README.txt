Trabalho sem fonte variável
========================

Este download contém trabalho sem as fontes variáveis e as fontes estáticas.

Work Sans é uma fonte variável com este eixo:
  wght

Isso significa que todos os estilos estão contidos nesses arquivos:
  Worksan-variablefont_wght.ttf
  Worksan-italic-variablefont_wght.ttf

Se o seu aplicativo suportar fontes variáveis, agora você pode escolher estilos intermediários
Isso não está disponível como fontes estáticas.Nem todos os aplicativos suportam fontes variáveis e
Nesses casos, você pode usar os arquivos de fonte estática para o trabalho, sem:
  estático/worksans-thin.ttf
  estático/worksans-exctraLight.ttf
  static/worksans-Light.ttf
  static/worksans-regus.ttf
  estático/worksans-medium.ttf
  estático/worksans-senibold.ttf
  estático/worksans-ous-bold.ttf
  estático/worksans-extrabold.ttf
  static/worksans-lack.ttf
  estático/worksans-thinitalic.ttf
  estático/worksans-exctraLightitalic.ttf
  estático/worksans-Lightitalic.ttf
  estático/worksans-italic.ttf
  estático/worksans-mediumitalic.ttf
  estático/worksans-semibolditalic.ttf
  estático/worksans-bolditalic.ttf
  estático/worksans-extrabolditalic.ttf
  static/worksans-blackitalic.ttf

iniciar
-----------

1. Instale os arquivos de fonte que você deseja usar

2. Use o seletor de fontes do seu aplicativo para ver a família de fontes e todos os
Estilos disponíveis
